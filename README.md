# 基于微信小程序的国产动漫论坛

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=30281

## 技术介绍
微信小程序+Java语言+ssm框架/springboot框架（后端）+Mysql(数据库)+Vue前端框架（Java Web）

## 开发工具
微信开发者工具（小程序端）+IDEA或者MyEclipse(后台管理端)+Navicat(mysql数据库可视化工具)

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/297%E5%A5%97-2-%E5%B0%8F%E7%A8%8B%E5%BA%8F%E5%BD%95%E5%83%8F/30281_weixin281%E8%AE%BA%E6%96%87%E5%BD%95%E5%83%8F.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
